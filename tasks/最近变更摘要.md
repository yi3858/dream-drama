# 最近变更摘要（v130 ~ v131）

## v131 — 为 AI 生成预览图添加提示词编辑弹窗

### 新增文件
- `supabase/functions/openclaw-proxy/index.ts` — OpenClaw 第三方 API 代理 Edge Function

### 修改文件
- `supabase/functions/generate-style-preview/index.ts`
  - 新增支持 `customPrompt` 自定义提示词参数
  - 优先使用管理员传入的自定义提示词，否则沿用自动构建的默认提示词

- `src/pages/admin/StylesPage.tsx`
  - 新增 `openPromptEditor` 函数：点击 AI 生成按钮时弹出提示词编辑框
  - 新增 `promptTarget`、`promptValue`、`promptDefault` 三个弹窗状态
  - 新增 `buildDefaultPrompt` 工具函数（与 Edge Function 保持一致的默认提示词构建逻辑）
  - `aiGeneratePreview` 函数新增可选参数 `customPrompt`
  - 卡片 hover 层及底部操作栏中，AI 生成按钮的 onClick 改为 `openPromptEditor`
  - 新增完整的「提示词编辑 Dialog」UI：
    - 顶部显示画风名称、描述、标签摘要
    - 大号多行文本编辑区（600字上限），预填默认提示词
    - 「重置为默认」按钮
    - 字符计数与提示词写作技巧说明
    - 取消 / 开始生成 按钮

### 效果
- 管理员点击「AI 生成预览图」按钮后，先弹出提示词编辑对话框
- 可自由修改提示词，或直接使用自动生成的默认提示词
- 点击「开始生成」后关闭弹窗，进入原有的 AI 生成进度流程

## v130 — 管理后台画风配置页添加 AI 自动生成预览图功能

### 新增文件
- `supabase/functions/generate-style-preview/index.ts` — AI 生成风格预览图 Edge Function

### 修改文件
- `src/pages/admin/StylesPage.tsx`
  - 新增 `aiGenMap` 状态：跟踪每个画风的 AI 生成状态（idle/generating/done/error）
  - 新增 `aiGeneratePreview` 函数：调用 Edge Function，轮询生成结果，更新数据库预览图 URL
  - 卡片 hover 操作层新增「AI 生成预览图」按钮（Sparkles 图标）
  - 卡片底部操作栏新增「AI 生成预览图」按钮（紫色边框）
  - 卡片预览区域新增 AI 生成进度遮罩（旋转动画 + "AI 生成中…" 提示）
  - 新增 AI 生成完成/失败提示遮罩

### Edge Function 功能
- 管理员鉴权（JWT + user_profiles.role = 'admin'）
- 自动根据画风 label/description/tags 构建 AI 图片生成提示词
- 调用图片生成接口提交任务，轮询最多 120 秒
- 生成成功后将图片转存至 `style-assets` Storage bucket
- 自动更新 `style_configs.preview_url` 字段
