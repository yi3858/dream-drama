# 需求文档

## 1. 应用概述

**应用名称**：AI漫剧制作聚合网站

**应用描述**：一个面向内容创作者的AI漫剧制作平台，提供小说文本一键生成漫剧、短剧实拍转动漫、广告视频制作、文生图、图生视频、动作迁移等核心能力，支持会员订阅、代理招商、作品管理等完整业务流程。采用简约科技风+漫剧二次元风格，深色主题,科技蓝紫渐变配色。

**技术架构**：Web网站，使用React + TypeScript + Tailwind CSS构建前端，Supabase作为后端数据库和认证服务，支持Docker一键部署，PC端优先并响应式适配手机端。

**对接能力**：阿里API、豆包API（用于AI生成能力）、阿里云内容安全API（用于合规审核）、RunningHub视频生成API（sparkvideo-2.0-fast，用于广告视频生成和图生视频）、RunningHub文生图工作流（用于文生图功能）、阿里云或字节跳动图生动作API（用于动作迁移功能）

**支付方式**：微信支付、支付宝支付

## 2. 用户与使用场景

**目标用户**：

- 小说作者、短剧创作者
- 内容运营团队
- 代理商、推广者
- 广告主、营销人员、门店经营者
- 设计师、插画师、视频创作者
- 动画制作者、特效师

**核心使用场景**：

- 创作者上传小说文本，经分镜确认后一键生成漫剧视频
- 创作者上传短剧实拍素材，转换为动漫风格并自动剪辑
- 用户使用角色库管理常用角色，提升创作效率
- 用户购买积分套餐按量使用AI生成服务
- 代理商推广平台获取返佣收益
- 用户查看作品案例、创作热点、爆款分析
- 广告主制作产品广告、品牌宣传片、带货短视频、门店推广、节日广告等视频内容
- 设计师通过文字描述生成图片素材
- 创作者将静态图片转换为动态视频
- 创作者上传人物/物体图片和动作参考视频，让图片主体复刻视频动作
- 用户通过邀请好友注册获得积分奖励
- 代理商申请代理资格，推广用户充值获取返佣积分，积分可用于平台消费或提现
- 用户批量提交多条文生图任务，统一管理生成进度
- 用户查看历史生成记录，回顾所有文生图和图生视频任务

### 3.2 功能说明

#### 3.2.1 首页

- 展示平台核心功能介绍（小说转漫剧、短剧转动漫、广告制作、文生图、图生视频、动作迁移）
- 展示作品案例精选
- 展示会员套餐入口
- 展示代理招商入口
- 提供快速开始创作入口
- 导航栏新增「广告制作」「文生图」「图生视频」「动作迁移」菜单项
- 首页新增「广告制作」「文生图」「图生视频」「动作迁移」功能卡片入口
- 动作迁移入口位于图生视频下方

#### 3.2.2 登录页面

- 支持用户名+密码登录
- 支持手机号+验证码登录
- 支持邮箱+密码登录
- 支持邮箱+验证码登录
- 邮箱密码登录表单下方增加「忘记密码？」链接
- 点击「忘记密码？」后弹出对话框，流程：输入邮箱 → 发送OTP验证码 → 验证通过 → 设置新密码 → 完成
- 提供注册入口

#### 3.2.3 注册页面

- 支持用户名+密码注册
- 支持手机号+验证码注册
- 支持通过邀请码/邀请链接注册并绑定邀请关系
- 注册成功后自动登录
- 完成手机号验证后自动赠送30积分

#### 3.2.4 小说转漫剧页面

- 用户上传小说文本文件或粘贴文本内容
- 提供「从角色库选取」按钮，弹窗展示用户已有角色，点击填入当前创作的角色列表
- 用户可手动填写角色名称和描述
- 提供画风选择器，展示预设风格模板（二次元风、写实风、3D国漫风等）
- 提供分辨率选择（720P、1080P、4K）
- 实时计算并展示预计消耗积分
- 弹出版权承诺协议勾选框，要求用户承诺拥有素材完整版权
- 提交后调用阿里云内容安全API进行敏感词过滤，检测不通过则拦截任务并提示违规原因
- 系统自动分镜（将文本拆解为场景片段）
- 跳转至分镜确认页面

#### 3.2.5 分镜确认页面

- 展示AI自动生成的分镜列表（场景顺序、台词、旁白）
- 支持手动调整场景顺序
- 支持修改单句台词或旁白
- 确认无误后提交进入视频合成阶段
- 调用阿里API或豆包API进行AI绘图生成分镜画面
- 自动合成视频并生成漫剧成品
- 所有生成的视频必须在画面显著位置强制添加「AI生成」水印
- 显示真实排队进度条及预估剩余等待时间
- 生成完成后触发站内信或短信通知用户
- 提供预览和下载

#### 3.2.6 短剧转动漫页面

- 用户上传短剧实拍视频素材
- 提供「从角色库选取」按钮，弹窗展示用户已有角色，点击填入当前创作的角色列表
- 用户可手动填写角色名称和描述
- 提供画风选择器，展示预设风格模板
- 提供分辨率选择（720P、1080P、4K）
- 实时计算并展示预计消耗积分
- 弹出版权承诺协议勾选框
- 提交后调用阿里云内容安全API进行违规画面过滤
- 调用阿里API或豆包API将实拍转换为动漫风格
- 自动剪辑视频（去除冗余片段、调整节奏）
- 自动生成字幕
- 所有生成的视频必须强制添加「AI生成」水印
- 显示真实排队进度条及预估剩余等待时间
- 一键成片并提供预览和下载

#### 3.2.7 广告制作页面（路由：/ad-maker）

**页面顶部提示说明**：

- 展示文字说明：「支持抖音、视频号、小红书、门店宣传屏等场景」

**广告类型选择**：

- 提供五种广告类型：产品广告、品牌宣传片、带货短视频、门店推广、节日广告

**提示词输入框**：

- 支持预设广告模板，根据选择的广告类型一键套用对应模板（每种类型2-3个预设模板）
- 可手动编辑修改提示词
- 字数限制500字

**分辨率与时长选择**：

- 分辨率选项：720P、1080P、4K
- 时长选项：5秒、8秒、10秒、15秒

**风格选择**：

- 提供六种风格：电影质感、高级质感、简约商务、国风、赛博朋克、写实
- 卡片式选择，每种风格附带简短描述

**上传素材**：

- 产品图片：支持JPG/PNG格式，单张≤10MB，可上传多张，最多3张

- 背景音乐：支持MP3/M4A格式，≤20MB，可选

**真人模式开关**：

- 开关控件，开启后适配真人出镜广告（realPersonMode=true）

**生成按钮与进度展示**：

- 弹出版权承诺协议勾选框
- 实时计算并展示预计消耗积分
- 积分不足时禁用生成按钮并引导充值
- 点击生成后显示实时进度条（轮询任务状态）
- 生成完成后展示视频预览
- 提供下载按钮
- 提示文字：「视频链接有效期24小时，请及时下载保存」
- 提示文字：「广告视频可用于抖音、视频号、小红书、门店宣传」

#### 3.2.8 文生图页面（路由：/text-to-image）

**模式切换**：

- 提供「单次模式」与「批量模式」开关
- 默认为单次模式

**单次模式**：

- 正向提示词输入框（必填，字数限制500字）
- 反向提示词输入框（可选，字数限制300字）
- 预设风格选择：卡片式选择，提供8种风格（写实摄影、二次元动漫、油画插画、赛博朋克、国风水墨、3D渲染、像素艺术、黑白素描）
- 图片比例选择：1:1、4:3、3:4、16:9、9:16
- 图片数量选择：1张、2张、4张
- 精细度选择：标准、精细、超精细
- 实时计算并展示预计消耗积分

- 积分不足时禁用生成按钮并引导充值

- 支持一键下载
- 支持复制提示词

**批量模式**：

- 提供添加提示词按钮，最多添加10条

- 支持删除单条提示词
- 全局参数设置：风格、比例、精细度、数量统一应用到所有任务
- 实时计算并展示总积分消耗汇总
- 积分不足时禁用提交按钮并引导充值
- 弹出版权承诺协议勾选框
- 提交后任务逐个顺序执行
- 每条任务显示独立进度状态（等待中/生成中/已完成/失败）
- 已完成的任务实时展示缩略图结果
- 支持单条任务单独下载
- 全部完成后提供一键下载全部图片按钮
- 失败的单条任务支持单独重试

## 4. 业务规则与逻辑

### 4.1 积分消耗规则

**小说转漫剧**：

- 基础消耗：每分钟视频消耗50积分
- 分辨率倍率：720P×1.0、1080P×1.5、4K×2.5

**短剧转动漫**：

- 基础消耗：每分钟视频消耗80积分
- 分辨率倍率：720P×1.0、1080P×1.5、4K×2.5

**广告制作**：

- 基础消耗：每秒视频消耗8积分
- 分辨率倍率：720P×1.0、1080P×1.5、4K×2.5
- 时长直接乘

**文生图**：

- 基础消耗：每张图3积分
- 精细度倍率：标准×1.0、精细×1.5、超精细×2.5
- 数量直接乘

**图生视频**：

- 基础消耗：每次生成10积分
- 时长倍率：3秒×1.0、5秒×1.5、8秒×2.2
- 分辨率倍率：480P×0.6、720P×1.0、1080P×1.8

**动作迁移**：

- 基础消耗：每次生成15积分
- 动作强度/贴合度不影响积分消耗

### 4.2 积分扣费与过期规则

**积分账户分离**：

- 数据库明确区分充值积分、赠送积分、返佣积分（冻结中）、返佣积分（可提现）
- 充值积分有效期按购买套餐档次计算（个人套餐永久有效，企业团队套餐90天有效）
- 赠送积分有固定有效期（默认30天）
- 返佣积分（冻结中）有固定冻结期（默认15天）
- 返佣积分（可提现）永久有效
- user_points_log积分流水表记录每次变动的来源、变动数值、积分类型、过期时间戳/解冻时间戳

**扣费优先级**：

- 优先扣除即将过期的赠送积分
- 再扣除返佣积分（冻结中）
- 最后扣除充值积分
- 返佣积分（可提现）不参与自动扣费，仅用于提现

**过期处理**：

- 超过有效期的积分自动清零

### 4.3 邀请机制规则

**邀请码生成**：

- 每个用户拥有唯一的邀请码
- 每个用户拥有唯一的邀请链接

**邀请奖励**：

- 新用户通过邀请码/邀请链接注册并完成手机号验证后，触发双向奖励
- 邀请人获得20积分（赠送积分，有效期30天）
- 被邀请人获得20积分（赠送积分，有效期30天）
- 新用户完成注册及手机号验证后，额外获得30积分（赠送积分，有效期30天）

**防刷机制**：

- 限制同一设备或IP的重复注册行为
- 只有真实有效的用户才能触发积分赠送

### 4.4 代理系统规则

**代理关系架构**：

- 平行独立架构：一级代理与二级代理之间不存在上下级隶属关系，互不产生级差分润
- 所有代理均直接隶属于平台，独立核算业绩

**代理等级与入门费**：

- 二级代理：入门费399元，享受8%返点
- 一级代理：入门费999元，享受15%返点

**返佣计算规则**：

- 返佣范围覆盖「推广充值」与「自身充值」两种场景
- 代理自己充值消费或下级用户充值，均按该代理当前等级对应的返点比例计算积分奖励
- 返佣积分计算公式：充值金额×积分汇率×返点比例
- 示例：二级代理推广用户充值100元，返佣积分=100×100×8%=800积分

**积分状态与消费逻辑**：

- 返佣积分分为「冻结中」与「可提现」两种状态
- 用户充值或自充产生的返佣积分，默认进入「冻结中」状态，冻结期为T+15天
- 冻结中积分允许直接用于平台内内容生成消耗（生成视频、图片），消费时优先扣除冻结积分
- 消费冻结积分等同于放弃该部分积分的提现权
- 仅当积分度过冻结期转为「可提现」状态后，方可申请提现

**提现规则**：

- 最低提现金额：500元（账户内需有5000积分处于「可提现」状态方可发起提现申请）
- 汇率标准：10积分兑换1元
- 提现申请提交后进入审核流程，审核通过后3-5个工作日内到账

**代理申请与审核**：

- 用户在代理申请页面选择代理等级并填写申请信息
- 提交申请后跳转支付页面，支付入门费
- 支付成功后提交代理申请，等待管理员审核
- 管理员确认入门费支付后开通代理权限

- 计算逻辑：（用户扣除积分÷积分汇率换算成金额）-实际API成本

**动态定价**：

- 积分消耗映射支持自动计算建议扣分：（单次调用成本×积分汇率）×（1+利润率）

**任务状态管理**：

- 每条任务独立状态：等待中/生成中/已完成/失败
- 已完成的任务实时展示缩略图结果
- 失败的单条任务支持单独重试，重试时仅扣除该任务对应积分

**结果下载**：

- 支持单条任务单独下载
- 全部完成后提供一键下载全部图片功能

### 4.19 生成记录规则

**记录来源**：

- 从generation_tasks表读取当前用户的所有历史记录

**筛选功能**：

- 支持按功能类型筛选（全部/文生图/图生视频）
- 支持按状态筛选（全部/生成中/已完成/失败）

**记录展示**：

- 每条记录显示时间、功能类型图标、渠道名称、状态徽章、积分消耗、提示词摘要
- 已完成的记录展示结果缩略图
- 支持点击缩略图放大预览
- 支持视频在线播放

**分页加载**：

- 支持分页展示，避免一次性加载过多数据

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 用户积分不足 | 禁用生成按钮，引导充值 |
| 上传文件格式不符 | 提示错误信息，拒绝上传 |
| 上传文件大小超限 | 提示错误信息，拒绝上传 |
| 内容审核不通过 | 拦截任务，提示违规原因 |
| 生成任务失败 | 返还消耗积分，提示失败原因 |
| 支付失败 | 提示支付失败，引导重新支付 |
| 网络异常 | 提示网络异常，引导重试 |
| API调用失败 | 提示生成失败，引导重试 |
| 视频链接过期 | 提示链接已过期，引导重新生成 |
| 用户未勾选版权承诺 | 禁用生成按钮，提示必须勾选 |
| 角色库角色数量达到上限 | 提示角色数量已达上限，引导删除或升级会员 |
| 代理提现金额不足 | 提示提现金额不足最低提现额度 |
| 同一设备或IP重复注册 | 拦截注册请求，提示异常行为 |
| 邀请码无效或已失效 | 提示邀请码无效，允许继续正常注册 |
| 管理员手动扣除积分 | 记录操作日志，更新用户积分余额 |
| 动作迁移上传视频时长超限 | 提示视频时长超限，拒绝上传 |
| 动作迁移参考图无法识别主体 | 提示图片无法识别主体，引导重新上传 |
| 套餐价格配置为空或异常 | 充值页面提示系统维护中，禁止购买 |
| 积分汇率配置为0或负数 | 系统拒绝保存，提示配置无效 |
| API成本设置为负数 | 系统拒绝保存，提示成本不能为负 |
| 邮箱已被其他用户绑定 | 提示邮箱已被占用，引导更换邮箱 |
| 邮箱验证码错误或过期 | 提示验证码错误或已过期，引导重新获取 |
| 忘记密码时邮箱未注册 | 提示邮箱未注册，引导注册或使用其他方式登录 |
| 利润率配置为负数或超过100% | 系统拒绝保存，提示利润率范围无效 |
| 代理申请入门费支付失败 | 提示支付失败，引导重新支付 |
| 代理申请被拒绝 | 提示拒绝原因，允许重新申请 |
| 代理提现申请被拒绝 | 提示拒绝原因，返还可提现积分 |
| 代理可提现积分不足5000 | 禁用提现按钮，提示积分不足 |
| 代理冻结积分被消费 | 扣除冻结积分，记录消费流水，该部分积分不再转为可提现 |
| 代理等级配置异常 | 系统拒绝保存，提示配置无效 |
| 返佣积分计算错误 | 记录错误日志，人工介入修正 |
| 批量任务提示词数量超过10条 | 禁用提交按钮，提示最多添加10条 |
| 批量任务中单条任务失败 | 该任务标记为失败状态，其他任务继续执行 |
| 批量任务提交后用户关闭页面 | 任务继续在后台执行，用户可在生成记录页面查看进度 |
| 生成记录页面加载失败 | 提示加载失败，引导刷新页面 |
| 生成记录中视频链接过期 | 提示链接已过期，引导重新生成 |

## 6. 验收标准

1. 用户通过邀请链接注册并完成手机号验证，获得30积分新人注册礼和20积分邀请奖励
2. 邀请人查看我的邀请页面，确认累计邀请人数+1，奖励明细中新增20积分记录
3. 用户进入我的钱包页面，确认总可用积分正确，充值积分、赠送积分、返佣积分（冻结中）、返佣积分（可提现）分开展示，即将过期的积分用醒目颜色标注
4. 用户使用积分生成作品，系统优先扣除即将过期的赠送积分，再扣除返佣积分（冻结中），最后扣除充值积分
5. 管理员进入推广积分设置页面，修改新人注册赠送积分数为50积分，保存后新用户注册获得50积分
6. 管理员进入邀请记录明细页面，按用户名筛选查看具体邀请关系链及积分发放情况
7. 管理员对疑似羊毛党用户进行备注并手动扣除违规积分，操作日志记录成功
8. 用户进入动作迁移页面，上传人物图片和动作参考视频，调整动作强度滑动条，点击开始生成，系统扣除15积分并显示进度条
9. 动作迁移任务完成后，用户预览生成的视频，确认图片主体复刻了参考视频的动作，点击下载保存视频
10. 管理员进入定价与成本管理-基础参数设置，修改积分汇率为1元=120积分，利润率为35%，保存成功
11. 管理员进入API成本设置子页面，编辑文生图功能的单次调用成本为0.05元，保存成功
12. 管理员进入积分消耗映射子页面，查看文生图基础消耗积分自动更新为建议值，手动修改并锁定为1积分，保存成功
13. 管理员进入套餐价格管理子页面，新增一个99元套餐并设置赠送比例，启用该套餐
14. 用户进入充值页面，确认展示的套餐信息与管理后台配置一致，选择99元套餐完成支付，积分到账
15. 管理员查看利润报表，确认某次文生图任务的利润计算正确：（用户扣除积分÷积分汇率）-实际API成本
16. 手机号注册用户进入个人中心-绑定邮箱页面，输入邮箱地址，点击发送验证码，输入验证码后绑定成功
17. 已绑定邮箱的用户在登录页面使用邮箱+密码方式登录成功
18. 用户在邮箱密码登录表单下方点击「忘记密码？」，弹出对话框，输入邮箱并发送OTP验证码，验证通过后设置新密码，完成密码重置
19. 用户进入代理申请页面，选择二级代理，填写申请信息，支付399元入门费，提交申请成功
20. 管理员进入代理审核页面，确认入门费支付成功，审核通过并开通代理权限
21. 代理用户进入代理中心页面，确认代理等级为二级代理，展示专属推广链接和邀请码
22. 代理用户复制推广链接分享给好友，好友通过链接注册并充值100元，代理用户获得800积分返佣（冻结中）
23. 代理用户进入推广记录页面，确认新增一条推广记录，显示好友充值金额100元，返佣积分800，状态为冻结中
24. 代理用户进入积分明细页面，确认新增一条返佣积分记录，显示积分来源为推广充值，积分状态为冻结中，解冻时间为15天后
25. 代理用户使用冻结中的返佣积分生成作品，系统扣除冻结积分，该部分积分不再转为可提现
26. 15天后，代理用户进入我的钱包页面，确认冻结中的返佣积分已转为可提现积分
27. 代理用户进入提现申请页面，填写提现金额500元（5000积分），填写收款信息，提交提现申请成功
28. 管理员进入提现审核页面，审核通过并标记已打款，代理用户收到提现到账通知
29. 代理用户进入提现记录页面，确认提现申请状态为已打款
30. 管理员进入代理管理-返佣配置页面，修改二级代理返点比例为10%，保存成功，新的推广充值按10%计算返佣
31. 管理员进入代理管理-全局结算配置页面，修改最低提现金额为1000元，保存成功，用户提现时需满足10000积分可提现
32. 管理员进入代理管理-积分流水页面，按用户筛选查看某代理的所有积分冻结/解冻/消费/提现记录
33. 用户进入文生图页面，切换到批量模式，添加5条提示词，设置全局参数（风格、比例、精细度、数量），系统实时计算总积分消耗汇总
34. 用户提交批量任务，系统一次性扣除全部积分，任务逐个顺序执行，每条任务显示独立进度状态
35. 批量任务中第3条任务失败，用户点击重试按钮，系统仅扣除该任务对应积分并重新生成
36. 批量任务全部完成后，用户点击一键下载全部图片按钮，成功下载所有生成的图片
37. 用户进入个人中心-生成记录页面，确认顶部统计卡片展示总生成次数、成功次数、消耗积分总计
38. 用户在生成记录页面选择「文生图」Tab，确认仅展示文生图相关记录
39. 用户在生成记录页面选择「已完成」状态筛选，确认仅展示已完成的记录
40. 用户点击某条已完成记录的缩略图，成功放大预览图片
41. 用户点击某条图生视频记录的视频封面，成功在线播放视频
