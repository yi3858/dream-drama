# 需求文档

## 1. 应用概述

**应用名称**：AI漫剧制作聚合网站

**应用描述**：一个面向内容创作者的AI漫剧制作平台，提供小说文本一键生成漫剧、短剧实拍转动漫、广告视频制作、文生图、图生视频等核心能力，支持会员订阅、代理招商、作品管理等完整业务流程。采用简约科技风+漫剧二次元风格，深色主题，科技蓝紫渐变配色。

**技术架构**：Web网站，使用React + TypeScript + Tailwind CSS构建前端，Supabase作为后端数据库和认证服务，支持Docker一键部署，PC端优先并响应式适配手机端。

**对接能力**：阿里API、豆包API（用于AI生成能力）、阿里云内容安全API（用于合规审核）、RunningHub视频生成API（sparkvideo-2.0-fast，用于广告视频生成和图生视频）、RunningHub文生图工作流（用于文生图功能）

**支付方式**：微信支付、支付宝支付

## 2. 用户与使用场景

**目标用户**：

- 小说作者、短剧创作者
- 内容运营团队
- 代理商、推广者
- 广告主、营销人员、门店经营者
- 设计师、插画师、视频创作者

**核心使用场景**：

- 创作者上传小说文本，经分镜确认后一键生成漫剧视频
- 创作者上传短剧实拍素材，转换为动漫风格并自动剪辑
- 用户使用角色库管理常用角色，提升创作效率
- 用户购买积分套餐按量使用AI生成服务
- 代理商推广平台获取返点收益
- 用户查看作品案例、创作热点、爆款分析
- 广告主制作产品广告、品牌宣传片、带货短视频、门店推广、节日广告等视频内容
- 设计师通过文字描述生成图片素材
- 创作者将静态图片转换为动态视频

## 4. 业务规则与逻辑

### 4.1 积分消耗规则

**小说转漫剧**：

- 基础消耗：每分钟视频消耗50积分
- 分辨率倍率：720P×1.0、1080P×1.5、4K×2.5

**短剧转动漫**：

- 基础消耗：每分钟视频消耗80积分
- 分辨率倍率：720P×1.0、1080P×1.5、4K×2.5

**广告制作**：

- 基础消耗：每秒视频消耗8积分
- 分辨率倍率：720P×1.0、1080P×1.5、4K×2.5
- 时长直接乘

**文生图**：

- 基础消耗：每张图3积分
- 精细度倍率：标准×1.0、精细×1.5、超精细×2.5
- 数量直接乘

**图生视频**：

- 基础消耗：每次生成10积分
- 时长倍率：3秒×1.0、5秒×1.5、8秒×2.2
- 分辨率倍率：480P×0.6、720P×1.0、1080P×1.8

### 4.2 会员权益规则

- 会员享受积分消耗折扣
- 会员享受优先生成队列
- 会员享受专属客服支持

### 4.3 代理返点规则

- 代理推广用户充值，代理获得返点收益
- 返点比例根据代理等级浮动
- 返点收益可提现

### 4.4 版权承诺规则

- 用户提交生成任务前必须勾选版权承诺协议
- 用户承诺拥有上传素材的完整版权
- 平台不承担用户素材侵权责任

### 4.5 内容审核规则

- 所有文本内容提交后调用阿里云内容安全API进行敏感词过滤
- 所有视频素材提交后调用阿里云内容安全API进行违规画面过滤
- 检测不通过则拦截任务并提示违规原因

### 4.6 水印规则

- 所有生成的视频必须在画面显著位置强制添加「AI生成」水印

### 4.7 任务队列规则

- 生成任务进入队列排队
- 会员用户享受优先队列
- 显示真实排队进度条及预估剩余等待时间

### 4.8 通知规则

- 生成任务完成后触发站内信或短信通知用户

### 4.9 文件存储规则

- 用户上传的素材存储至Supabase Storage
- 广告制作上传的素材存储至ad-assets bucket
- 图生视频上传的参考图存储至creation-assets bucket或复用ad-assets bucket
- 生成的图片通过URL直接展示，不存储至数据库

### 4.10 API对接规则

**小说转漫剧与短剧转动漫**：

- 调用阿里API或豆包API进行AI绘图和视频生成

**广告制作**：

- 调用RunningHub视频生成API（sparkvideo-2.0-fast）
- 通过Edge Function后端调用，API Key不暴露前端

**文生图**：

- 调用RunningHub文生图工作流
- 通过Edge Function后端调用，API Key不暴露前端

**图生视频**：

- 调用RunningHub sparkvideo-2.0-fast工作流（图生视频模式）
- 通过Edge Function后端调用，API Key不暴露前端

**内容审核**：

- 调用阿里云内容安全API进行敏感词过滤和违规画面过滤

**行业数据**：

- 对接第三方行业数据平台（如DataEye、灰豚数据等）获取全网漫剧热度榜和题材趋势数据

### 4.11 支付规则

- 支持微信支付和支付宝支付
- 支付成功后自动充值积分到账户
- 积分永久有效（企业团队大额套餐除外，有效期90天）

### 4.12 角色库规则

- 用户可创建私有角色或公开角色
- 公开角色展示在公共角色库供所有用户使用
- 用户可收藏公共角色到个人角色库
- 角色名称最多20字，描述最多200字，标签最多5个

### 4.13 视频链接有效期规则

- 广告视频链接有效期24小时
- 图生视频链接有效期24小时
- 用户需及时下载保存

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 用户积分不足 | 禁用生成按钮，引导充值 |
| 上传文件格式不符 | 提示错误信息，拒绝上传 |
| 上传文件大小超限 | 提示错误信息，拒绝上传 |
| 内容审核不通过 | 拦截任务，提示违规原因 |
| 生成任务失败 | 返还消耗积分，提示失败原因 |
| 支付失败 | 提示支付失败，引导重新支付 |
| 网络异常 | 提示网络异常，引导重试 |
| API调用失败 | 提示生成失败，引导重试 |
| 视频链接过期 | 提示链接已过期，引导重新生成 |
| 用户未勾选版权承诺 | 禁用生成按钮，提示必须勾选 |
| 角色库角色数量达到上限 | 提示角色数量已达上限，引导删除或升级会员 |
| 代理提现金额不足 | 提示提现金额不足最低提现额度 |

## 6. 验收标准

1. 用户注册并登录账户
2. 用户充值积分
3. 用户进入文生图页面，输入提示词，选择风格和参数，生成图片并下载
4. 用户进入图生视频页面，上传参考图片，输入运动描述，选择参数，生成视频并下载
5. 用户进入小说转漫剧页面，上传小说文本，选择画风和分辨率，确认分镜，生成漫剧视频并下载
6. 用户进入广告制作页面，选择广告类型，输入提示词，上传素材，生成广告视频并下载
7. 用户查看个人中心的作品管理，确认所有生成的作品均已保存
